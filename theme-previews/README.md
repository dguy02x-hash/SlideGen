# Theme Preview Images

## How to Add Theme Preview Images

1. Navigate to the folder for your theme (e.g., `Business Black and Yellow/`)
2. Add your preview images (PNG, JPG, or GIF)
3. Name them sequentially: `preview1.png`, `preview2.png`, `preview3.png`, etc.
4. The images will automatically appear in the theme selector

## Folder Structure

```
theme-previews/
  /Business Black and Yellow/
    - preview1.png
    - preview2.png
    - preview3.png
  /Simplistic Red and White/
    - preview1.png
    - preview2.png
  /Ocean Blue/
    - preview1.png
  ...
```

## Image Guidelines

- **Format**: PNG, JPG, or GIF
- **Recommended Size**: 800x600px or 1024x768px
- **File Names**: Must be named `preview1.png`, `preview2.png`, `preview3.png`, etc.
- **Content**: Screenshots of slides using that theme
- **Multiple Images**: Add as many preview images as you want for each theme

## Adding a New Theme

1. Create a new folder with the exact theme name
2. Add preview images to that folder
3. The theme will automatically appear in the selector
