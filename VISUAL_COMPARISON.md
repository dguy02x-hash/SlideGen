## Visual Pattern Recognition

### Meta-Instruction Indicators (REMOVE):
- Any phrase starting with "Make sure..."
- Any phrase starting with "Pay attention..."
- Any phrase starting with "Don't forget..."
- Any phrase containing "is crucial"
- Any phrase containing "is important"
- Any phrase containing "you need to"
- Any phrase containing "the key thing"
- Phrases that tell someone HOW to present

### Good Explanatory Indicators (KEEP):
- Dates and historical context
- Cause and effect relationships ("because", "driven by", "led to")
- Comparisons and contrasts
- Specific statistics and data
- Real-world examples
- Natural transitions between ideas
- Information that explains WHAT, not instructions on HOW to present
